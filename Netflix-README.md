# Netflix Data Analysis Dashboard

## Overview
This project analyzes Netflix's catalog of movies and TV shows to identify trends in content type, release years, and global distribution.

The dashboard was built using Google Sheets, focusing on data cleaning, analysis, and visualization.


## Tools Used
- Google Sheets
- Pivot Tables
- Data Visualization (Charts)
- Basic Data Cleaning


## Dataset
Netflix Movies and TV Shows dataset (sourced from Kaggle)


## Key Metrics
- Total Titles
- Total Movies
- Total TV Shows
- Number of Countries


## Analysis Performed
- Distribution of Movies vs TV Shows
- Number of titles released over time
- Content distribution by genre
- Country-level content production


## Key Insights
- Netflix content significantly increased after 2015
- Movies make up the majority of the platform
- TV Shows have been growing steadily in recent years
- Content is produced across multiple countries, with a few dominating production
