# Sales Data Analysis Project

## Overview
This project analyzes sales data using Google Sheets to identify trends in sales, customers, and categories.

## Tools Used
- Google Sheets 
- Pivot Tables
- Charts

## Key Insights
- Total sales calculated across all transactions
- Sales analyzed by category and region
- Customer count determined using unique IDs
